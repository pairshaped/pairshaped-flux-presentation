
# Pairshaped Flux Presentation

## Summary

An example Todo Application with Coffeescript and React, with and without Flux.
Includes presentation on the advantages of using Flux instead of straight up React.

## Interesting places to look:

 * 0-todo-noflux
  * What a multi-list todo app looks like without flux
 * 1-todo-flux
  * What a multi-list todo app looks like with flux
 * presentation
  * Keynote, PDF, and stuff!
